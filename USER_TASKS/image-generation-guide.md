# La Tribu — Image Generation Guide

> **How to use:** Copy the **Brand Context** section into ChatGPT at the start of a conversation. Then paste each image prompt one at a time. Download each result and rename to the exact filename listed.

---

## Brand Context (paste this first)

```
You are generating photorealistic images for La Tribu, a Latin nightlife brand in London. Every image must feel like it belongs in the same visual world.

COLOUR PALETTE (use these as lighting, accents, and atmosphere — not as flat backgrounds):
- Deep charcoal black: #12100e — dominant, the darkness of a nightclub
- Coral red: #ef5a3f — warm accent lighting, neon glows, reflections
- Magenta pink: #d92c69 — secondary accent, edge lighting, haze tints
- Gold amber: #e2a23c — warm highlights, liquid tones, candlelight
- Cream: #f4eadc — soft highlights, smoke diffusion, fabric textures

MOOD: Sexy, warm, cinematic. Think high-end nightlife photography — shallow depth of field, dramatic lighting, rich shadows. The images should feel expensive and alive. Moody but inviting. Dark backgrounds with pops of coral, magenta, and gold light.

STYLE: Photorealistic. Shot on a full-frame camera with a fast prime lens (50mm f/1.4 or 85mm f/1.4). Shallow depth of field. Warm colour grading. No text, no watermarks, no logos, no people, no faces, no hands.

ABSOLUTELY NO PEOPLE in any image. No faces, silhouettes, body parts, or crowds. Focus on objects, textures, drinks, instruments, spaces, and light.
```

### 20. Logo — `logo.webp` + `logo-white.webp`

> - `logo-white.webp`: cream text (#f4eadc) for dark backgrounds

## After Generating

1. Download each image from ChatGPT
2. Rename to the exact filename listed above (keep `.webp` extension — ChatGPT may give you `.png`, that's fine, just rename)
3. Place all files in the `images/` folder
4. Commit and push:

```bash
cd ~/projects/latino-events
git add images/
git commit -m "Add generated brand images"
git push
```

The site will auto-deploy with the new images if Vercel GitHub integration is connected.
